Symbol

[edit]

42 (previous 41, next 43)

  1. The cardinal number forty-two.

Usage notes

[edit]

The number 42 is often given as a joking answer to the question of the meaning of life, the universe, and everything, and is sometimes used as an example value in programming where a number is desired, in reference to The Hitchhiker's Guide to the Galaxy.

For quotations using this term, see Citations:42.
